# statix-agent

Project layout:

- `statix/spec.yaml` is the canonical Statix V2 project spec.
- `statix/tasks/`, `statix/env/`, `statix/deploy/`, and `statix/policies/` contain split V2 spec fragments.
- `statix/` contains deployment and Statix-managed files.
- `app/` contains the application source.
